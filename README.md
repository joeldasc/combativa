# Combativa

## Pre Requesitos

- Contar con [Node.js](https://nodejs.org/es/) - Versión >= 14
- Contar con [NPM](https://yarnpkg.com/) - Versión >= 6

```BASH
npm install
```

### Rutinas útiles

- Iniciar para levantar el proyecto en local

```BASH
npm run start
```

- Generar build de prueba

```BASH
npm run build
```

- Generar archivos para producción

```BASH
npm run dist
```
